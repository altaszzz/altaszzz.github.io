560. Subarray Sum Equals K

Given an array of integers and an integer k, you need to find the total number of continuous subarrays whose sum equals to k.

Example
Input: nums = [1,1,1], k = 2
Output: 2

In the solution above, we have a hash map, the key of the hash map is sum of first i elements. the value in the hash map is the occurences of the sum.

Note we have to put m[0]=1 in the hash map.
Why? The idea of having the hash map is to find if sum - k exist or not. if sum == k, then the sum from the first index to current index will be k. so we make m[0] = 1

523. Continuous Subarray Sum

Math : if a % k = x and b % k = x -> (a - b) % x = 0

Note in this problem, we have to make sure the size of the sub-array has to be greater and equal than 2, so we have a hash map here to remember the index

525. Contiguous Array

Need a trick to calculate the sum of first i elements
When meeting a 1, we add 1 to the sum, when meet a 0, we add -1 to the sum.
so when sum(i) == sum(j) that’s been said between i and j (in-inclusive), we have same amount of 1 and 0

461. Hamming Distance

Description
example:
x: 0 0 0 1
y: 0 1 0 0

x ^ y : 0 1 0 1
after doing xor, we will get all distinct 1s
then we could do val & val - 1

477 Total Hamming Distance

example:
4: 0 1 0 0
14: 1 1 1 0
2: 0 0 1 0
1: 0 0 0 1

let’s look at the last digit, we got 0 0 0 1, so the hamming distance is 3
Then look at the 3rd digit, we got 0 1 1 0, each 1 has 2 hamming distance with 0, so total is 4. The 2nd digits is also 4, the first digit is 1
In total it’s 3 + 4 + 4 + 3 = 20
